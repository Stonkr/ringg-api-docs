## Prerequisites

Before starting the integration, ensure you have:

- Exotel account with API access
- Account SID and API Key/Token from Exotel dashboard
- At least one Exotel phone number (ExoPhone)
- Account on Ringg

## Step 1: Exotel Account Setup

### Create an Exotel Account

1. Sign up at [Exotel Dashboard](https://my.exotel.com)
2. Complete KYC verification (required for Indian accounts)
3. Add credits to your account

### Obtain API Credentials

Navigate to Settings → API Settings in your Exotel dashboard:

<CodeGroup>
```bash Credentials
ACCOUNT_SID=example1234
API_KEY=a1b2*********ci9j0
API_TOKEN=z9y************t3s2r1q0
SUB_DOMAIN=api.exotel.com
APP_ID=345*******4234
```
</CodeGroup>


## Step 2: ExoPhone Configuration

### Provision an ExoPhone

1. Go to **ExoPhones** in your Exotel dashboard
2. Click **Buy Number** or **Rent Number**
3. Select your preferred region (Delhi, Mumbai, Bangalore, etc.)
4. Complete the purchase

## Step 5: Configure Inbound Calls

### Attach Numbers to AI Assistants

<Steps>
  <Step title="Go to Numbers Management">
    Navigate to **Numbers** → **Manage Numbers** in your dashboard
  </Step>
  <Step title="Select Your Exotel Number">
    Click on the Exotel number you want to configure
  </Step>
  <Step title="Assign AI Assistant">
    Choose which AI assistant should handle inbound calls to this number
  </Step>
  <Step title="Configure Call Settings">
    Set up:
    - Call recording preferences
    - Transcription options
    - Webhook notifications
  </Step>
  <Step title="Save Configuration">
    Click **"Save"** to apply your settings
  </Step>
</Steps>
