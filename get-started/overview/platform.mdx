## What is Ringg AI?

Ringg AI is an AI voice agent platform built for businesses that need to automate phone-based interactions at scale. Whether you're following up on loan collections, qualifying leads, onboarding customers, or handling inbound support — Ringg AI's voice assistants handle the conversation for you.

---

## Key Capabilities

| Capability | Detail |
|---|---|
| Concurrent Calls | 10,000+ simultaneous calls |
| Script Adherence | 99.9% accuracy |
| Languages Supported | 20+ languages |
| Call Types | Outbound, Inbound, Webcall |
| Industries | Finance, Logistics, Healthcare, Education, eCommerce, HR |

## How It Works

<Steps>
  <Step title="Create an Assistant">
    Define your AI voice agent's name, language, voice, and prompt (script/objective).
  </Step>
  <Step title="Add a Number">
    Purchase or bring your own telephony number.
  </Step>
  <Step title="Launch a Campaign">
    Upload a calling list, set a schedule, and let the assistant make calls automatically.
  </Step>
  <Step title="Review Results">
    Analyse call transcripts, latency metrics, and outcome data in History and Analytics.
  </Step>
</Steps>

## Platform Architecture

The Ringg AI dashboard is organized into the following sections:

**App**
- **Assistants** — Create and configure AI voice agents
- **Campaigns** — Schedule and manage bulk outbound call campaigns
- **Knowledge Base** — Upload documents or FAQs for assistants to reference
- **Numbers** — Manage your telephony numbers
- **History** — Review call logs and transcripts
- **Analytics** — Monitor performance and call metrics

**More**
- **Workspace Members** — Invite team members and manage permissions
- **API Key** — Access your API credentials for programmatic integrations
- **Integrations** — Connect external telephony providers and tools like Zapier
